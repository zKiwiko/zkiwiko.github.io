Aim Assist

General Difficulty: ★ ★

Here are some curated examples of 'Aim Assist' enhancing mods that can be used in various FPS games.

Basic Example

define RADIUS = 10; define SPEED = 20; define AIM = XB1_LT; define RY = XB1_RY; define RX = XB1_RX; main { if(get_val(AIM)) { combo_run(AimAssist); } } combo AimAssist { set_val(RY, inv(RADIUS + get_ival(RY))); wait(SPEED); set_val(RX, RADIUS + get_ival(RX)); wait(SPEED); set_val(RY, RADIUS + get_ival(RY)); wait(SPEED); set_val(RX, inv(RADIUS + get_ival(RY))); }
